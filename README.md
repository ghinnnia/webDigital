# Web Digital Agency

A comprehensive management system for a Digital Agency built with Laravel.

## Features

- **Employee Management**: Track employees, roles, and divisions.
- **Attendance & Leave**: Manage employee attendance, leave requests (Cuti), and medical certificates.
- **Project & Task Management**: Track projects, assign tasks, and monitor progress.
- **Finance**: Handle orders, invoices, and cashflow.
- **Announcements & Meeting Notes**: Share information across the company.

## Technology Stack

- **Backend**: Laravel
- **Frontend**: Blade, JavaScript (Vite)
- **Database**: MySQL

## Installation

1. Clone the repository.
2. Run `composer install`.
3. Run `npm install` and `npm run dev`.
4. Configure `.env` file (Database, etc.).
5. Run `php artisan migrate`.
6. Run `php artisan serve`.
